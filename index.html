About TubeSumTalk

TubeSumTalk is a powerful browser extension that enhances your YouTube viewing experience by providing instant video summaries and enabling AI-powered conversations about the content. Save time and gain deeper insights from your favorite videos.

Key Features

Smart Summaries

Generate concise summaries of YouTube videos in various formats: bullet points, brief overviews, or detailed analyses.

AI Chat

Engage in meaningful conversations about video content with our advanced AI powered by Google's Gemini model.

Customizable

Choose your preferred summary length and style to match your reading preferences.

Easy to Use

Seamlessly integrated into your YouTube experience with a clean and intuitive interface.

Installation Instructions

  1. Clone the repository or download the latest release from GitHub
  2. Open your browser's extension management page
  3. Enable Developer Mode
  4. Click "Load unpacked" and select the extension directory
  5. Configure your API keys in the extension settings

How to Use

  1. Navigate to any YouTube video
  2. Click the TubeSumTalk extension icon
  3. Choose your desired summary type and length
  4. Get instant summaries and start chatting about the content

Technology Stack

  • JavaScript (ES6+)
  • Chrome Extension APIs
  • Google's Gemini AI Model
  • Node.js Backend
  • FastAPI & Flask
